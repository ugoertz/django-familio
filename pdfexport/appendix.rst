Verwendete Software
---------------------------

Die Webseite der Familienchronik und die pdf-Export-Funktion wurden mit einer
Vielzahl von Open-Source-Softwarepaketen und -Projekten programmiert, unter
anderem:

`Python <http://www.python.org>`__,
`Django <http://www.djangoproject.com>`__,
`jQuery <http://jquery.com/>`__

Karten
......

`OpenStreetMaps <https://www.openstreetmap.org/about>`__,
`Leaflet <http://leafletjs.com/>`__,
`Wikidata <http://www.wikidata.org>`__,
`Mapnik <http://mapnik.org/>`__,
`TileStache <http://tilestache.org/>`__

Ahnentafeln, Nachkommen
.......................

`D3.js <http://d3js.org/>`__

Schriften, Icons, Layout
........................

`Vollkorn <http://vollkorn-typeface.com/>`__,
`Bootstrap <http://getbootstrap.com/>`__
(mit `Glyphicons <http://glyphicons.com/>`__),
`Font Awesome <http://fontawesome.io/>`__

PDF-Export
..........

`Sphinx <http://sphinx-doc.org/>`__,
XeLaTex,
`ImageMagick <http://www.imagemagick.org/>`__


Verwaltungsinterface
....................

`CodeMirror <http://codemirror.net/>`__


Entwicklung, Tests
..................

`VIM <http://www.vim.org/>`__,
`Git <http://git-scm.com/>`__,
`Selenium <http://www.seleniumhq.org/>`__

Außerdem verwendet wurde eine große Anzahl von Python- und Django-Paketen:
celery, grappelli, watson, userena, django-markup, reversion, dajaxice,
django-leaflet, django-geojson, django-braces, pybbm, pillow, cairocffi,
ipython, Werkzeug, django-debug-toolbar, django-extensions, ...
